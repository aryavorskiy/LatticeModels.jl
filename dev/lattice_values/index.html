LatticeValue basics

A LatticeValue is a struct that maps sites of a certain Lattice to values of some type. One can be generated using a do-syntax similar to one in Sublattices:

julia> l = SquareLattice(10, 10)100-site square lattice (10×10 macrocell)
julia> lv = LatticeValue(l) do (x, y); x + y + 1; end    # arbitrary site-dependentLatticeValue{Float64, :square} on a 100-site square lattice (10×10 macrocell)

To generate a tuple of LatticeValues for site coordinates, you can use the coord_values function. Note that LatticeValues support broadcasting, which means you can create coordinate-dependent lattice values in-place:

julia> lv == x .+ y .+ 1true
Note

A wavefunction cannot and must not be stored as a LatticeValue, use LatticeVector instead. The reason is that a LatticeValue does not take one-site phase spaces into account, which renders them unusable for these purposes.

Linear algebra operations and @on_lattice wrapping are deliberately unsupported for LatticeValues.

Lattice values implement a scatter plot recipe, which colors the plot markers according to the value:

scatter(lv, markersize=10)

Depending on the lattice type, additional plot recipes can be available. For example, a lattice value on a square lattice can be plotted as a heatmap (which will be enabled by default if you do not specify the series type):

plot(lv, markersize=10)

Indexing

It is often required to select some sites by certain condition. This can be done using a LatticeValue{Bool} and broadcasting (like with Sublattices).

In the example below we will delete all sites from the circle of radius 3 in the center of the lattice (which will make the according heatmap regions blank).

heatmap(lv[@. √(x^2 + y^2) ≥ 10])

The approach from above provides a flexible way to edit LatticeValues:

lv2 = ones(l)
lv2[@. x < y] = x .* y          # Assign another LatticeValue
lv2[@. x ≥ y && x ≥ -y] .= 20   # or a number
heatmap(lv2)

Note that a LatticeValue can be projected to some coordinate axis to create line plots. The projection axis is set by an axis descriptor (see Axis descriptors)

lv_on_line = lv[x = 5]
p = plot(layout=(2, 1))

heatmap!(p[1], lv)
plot!(p[1], lattice(lv_on_line), high_contrast=true)
plot!(p[2], project(lv_on_line, :y))

Note that we can show the sites we selected by plotting the lattice of the selected values with high_contrast=true. This options hides the indices and translucent marks, and also makes the plot markers black-and-white, which prevents them from blending in with the heatmap in the background.
